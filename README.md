# AUSTRA Web

Sitio Web Austra

descripcion

El sitio presenta una marca de ropa inspirada en la Patagonia argentina.

Incluye:

- Home con presentación de la temporada y categorías de producto.
- Página de colección general (`productos.html`) con remeras, anoraks, buzos y rompevientos.
- Página específica de accesorios y branding (`accesorios.html`).

La estética está basada en una paleta tierra: verde vejiga, ocre oscuro, gris humo, negro y blanco.



 https://federovsilzle.github.io/web-austra/

Herramientas usadas

- HTML5 semántico
- CSS3 (variables, media queries y efectos hover)
- Bootstrap 5 (grid, navbar, cards, accordion)
- Git y GitHub para control de versiones
- GitHub Pages para el despliegue
